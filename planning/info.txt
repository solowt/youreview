User Stories:

1.  As a user, I want to be able to review movies so that other people can read my opinions about things.

2.  As a user, I want to be able to review music so that other people can read my opinions about things.

3.  As a user, I want to be able to review books so that other people can read my opinions about things.

4.  As a user, I want to be able to get a list of tracks on the album I am reviewing so that I don't have to get that information myself.

5.  As a user, I want to be able to rate reviews based on their helpfulness so that other users will be able to avoid bad reviews.

6.  As a user, I want to be able to leave comments on reviews so that review authors can get feedback.

7.  As a user, I want to be able to generate a list of books by the author whom I am reviewing so that I don't have to find other books myself.
